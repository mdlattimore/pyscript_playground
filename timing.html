Code Timer

 

This is simple code timer that measures (in milliseconds) how long it takes code running in the browser to count from 0 to 1,000,000. The timer begins when you click one of the "Go!" buttons. The top button executes a JavaScript function. The bottom button executes an equivalent function written in Python, running within the PyScript framework (which allows Python to be run directly in the browser).

Python Function

 

import time def py_run(*args, **kwargs): start = time.perf_counter() x = 0 while x < 1000000: x += 1 stop = time.perf_counter() timed = f"{(stop - start) * 1000}" display = f"{timed} milliseconds." Element('py_result').write(display)
